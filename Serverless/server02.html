    # 什么是云开发（小程序）

    小程序云开发是微信团队联合腾讯云提供的原生 Serverless 云服务

    开发者可以使用云开发开发微信小程序、小游戏，无需搭建服务器，即可使用云端能力
    云开发为开发者提供完整的原生云端支持和微信服务支持，弱化后端和运维概念，无需搭建服务器，使用平台提供的 API 进行核心业务开发，即可实现快速上线和迭代，同时这一能力，同开发者已经使用的云服务相互兼容，并不互斥。

    # 云开发能力介绍

    • 存储： 在小程序端直接上传/下载云端文件，可视化管理
    • 云函数： 在云端运行的代码，微信私有天然鉴权，开发者只需要编写自身业务逻辑代码
    • 云数据库： 一个即可在小程序前段操作，也能在云函数中读写的JSON数据库
    • 运营分析：查看云开发监控、配额使用量、用户访问情况

    其他：

    • 音视频服务：提供互通高品质实时音视频通话服务，美颜滤镜，高清视频通话
    • 智能图像服务：集成智能鉴黄、人脸识别、人脸核身等AI视觉能力
